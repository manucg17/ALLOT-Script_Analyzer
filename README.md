# ALLOT-Script_Analyzer
Script_Analyzer for ALLOT
# Setup and Configuration

## Requirements
- Python 3.x
- Flask
- Werkzeug

## Setup Instructions

### 1. Install Python 3.x
If you don't have Python installed, download and install it from the official Python website: Version 3.12.1 or above

### 2. Install Flask and Werkzeug
pip install Flask
pip install Werkzeug

### 3. To run the Application:
python app.py

### 4. To Access the Applicaton:
Open a web browser and go to http://localhost:5000
If the IP of the PC is accessible within a private or public network, the GUI can be accessed with the IP of the Server this application is hosted on:
- Assume the Server IP: 192.168.0.79, Access it via  http://192.168.0.79:5000/
